Tftp服务器

在嵌入式开发过程中有宿主机和目标机的角色之分：宿主机是执行编译、链接 嵌入式软件的计算机；目标机是运行嵌入式软件的硬件平台

Tftp服务器-安装

1、如果利用如下命令能够看到服务已启动, 则不用安装 [arm@localhost]# netstat -a | grep tftp

已安装结果： udp 0 0 *:tftp *:*

2、如果没有安装，执行如下命令安装。

[root@localhost]# rpm -ivh tftp-server-0.423.1.i386.rpm(rhel5安装光盘\Servers目录)

3、建立tftp的主工作目录

[root@localhost btools]# mkdir /tftpboot

Tftp服务器-配置

4、修改配置文件

[root@localhost btools]# vi /etc/xinetd.d/tftp

service tftp

{

   …………………………

   socket_type = dgram

   …… …… ……

   server = /usr/sbin/in.tftpd

   server_args = -s /tftpboot

   disable = no

   …… …… ……

}

Tftp服务器-启动

#/etc/init.d/xinetd start

#netstat -a | grep tftp

udp 0 0 *:tftp *:*
